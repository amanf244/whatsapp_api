## Description

simple whatsapp api

Postman api [click here](https://www.getpostman.com/collections/8b3e718cc4773c937ccf).

## Installation

```bash
$ npm install
# if usimg yarn
$ yarn install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
